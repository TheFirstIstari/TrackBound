# Contributing

Thanks for your interest in contributing to TrackBound! A few guidelines to get you started.

How to contribute
- Fork the repo and create a branch named `feature/your-short-desc` or `fix/short-desc`.
- Run tests and linters locally before opening a PR (commands below).
- Open a focused PR with a clear description and link any relevant issue.

Local workflows

Run tests:

```bash
flutter test
```

Format code:

```bash
flutter format .
```

Lint:

```bash
flutter analyze
```

PR checklist
- [ ] Tests added/updated
- [ ] Code formatted (`flutter format`)
- [ ] Lint issues addressed (`flutter analyze`)
- [ ] PR description explains the why and what

Communications
- For sensitive issues or CODE_OF_CONDUCT violations, contact the maintainers at: your-email@example.com
